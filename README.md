A classic and interactive Hangman game built with **Streamlit**. Guess the word by entering letters before the hangman is complete. Fully styled with a modern glassmorphism UI and real-time feedback.  

## Features
- Random word selection from a predefined list  
- Tracks guessed letters (correct and incorrect)  
- Shows remaining attempts (6 chances)  
- Visual feedback for correct/wrong guesses  
- Stylish Streamlit interface with gradients, shadows, and glass cards  
- Replay / Restart option  

## How To Play
1. Run `streamlit run hangman.py` in your terminal.  
2. Enter one letter at a time in the input box.  
3. Correct guesses reveal letters in the word.  
4. Wrong guesses decrease your attempts.  
5. Guess the word before attempts reach 0 to win.  
6. Click "Restart Game" to play again.
